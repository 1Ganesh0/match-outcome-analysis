# IPL Match Outcome Analysis

## Objective

Analyze IPL match data to identify factors influencing match outcomes.

## Methodology

* Data cleaning and preprocessing
* Feature engineering (powerplay, death overs, total runs)
* Exploratory Data Analysis
* Correlation and statistical insights

## Key Insights

* Strong powerplay improves win probability
* Total runs have the highest impact
* Death overs alone are not decisive
* Balanced performance across phases is critical

## Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn

## Conclusion

Match outcomes in IPL are influenced by multiple factors rather than a single dominant metric.
